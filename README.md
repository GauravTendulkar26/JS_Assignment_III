# JS_Assignment_III

1. Create a discount calculator that calculates the discounted price of an item based on
the original price and discount percentage. Allow users to input the original price and
discount percentage and display the discounted price.
2. Implement a number guessing game where the computer generates a random
number between 1 and 100, and the user has to guess the number. Provide
feedback to the user if their guess is too high or too low.
3. Write a program that calculates the age of a person based on their birth year. Allow
users to input their birth year and display their age.
4. Create a simple student management system using JavaScript objects. Allow users
to add new students, display student information, and delete students.
5. Write a program that accepts input from the user and determines the type of the input
(number, string, Boolean, array, object, etc.).
6. Create a BMI (Body Mass Index) calculator. Allow users to input their weight (in
kilograms) and height (in meters) and calculate their BMI. Use the formula: BMI =
weight / (height * height).
7. Create a simple calculator that performs addition, subtraction, multiplication, and
division. Allow users to input two numbers and select the operation to perform.
8. Write a program to convert temperature from Celsius to Fahrenheit and vice versa.
Allow users to input the temperature and the desired unit of measurement (Celsius or
Fahrenheit).
